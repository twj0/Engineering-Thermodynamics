## 10.2 再热循环
- 吸热量 $q_1 = (h_1 - h_4) + (h_a - h_b)$
- 放热量 $q_2 = (h_2 - h_3)$
- 净功（忽略泵功） $w_{net} = (h_1 - h_b) + (h_a - h_2)$
- 热效率 $\yita_{t,RH} = \frac {w_{net}}{q_1} = \frac {(h_1 - h_b) + (h_a - h_2)}{(h_1 - h_4)+(h_a - h_b)}$

### 蒸汽抽汽回热的特点
- 优点
	- 提高热效率
	- 减小汽轮机低压缸尺寸，末级叶片变短
	- 减小凝汽器尺寸，减小锅炉受热面
- 缺点
	- 循环比功减小，耗汽率增加
	- 增加循环设备复杂性
	- 回热器投资
