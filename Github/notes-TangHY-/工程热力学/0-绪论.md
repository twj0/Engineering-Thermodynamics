# 0. 绪论
## 1. 什么是工程热力学
- 热力学：研究能量机器相互转化规律的科学
- 工程热力学：从工程技术观点出发，研究热能有效利用以及和其他形式能量转换规律及工程应用的科学
## 2. 能源定义
- 凡是能直接或者转换而获取某种能量的自然资源统称为能源
- 能源可以根据来源、形态、使用程度和技术、污染程度以及性质进行分类
	- 能量来源
		- 天外：宇宙射线、太阳能、水力、风、……
		- 本身：核能、地热能
		-地球与其他天体相互作用：潮汐能
	- 能源存在形态
		- 一次：可直接利用
			- 可再生：风能、水能、地热能
			- 非再生：煤、石油、天然气
		- 二次：由一次能源直接或间接转化
			- 电、高温蒸汽、焦炭、煤气
